1.3 Game Description

Trade Clash is a browser-based sealed-bid auction game where:

  • 2-4 players compete in real-time auction rooms on a 3D globe
  • Each lot bundles 7 real prediction market positions with a hidden true value
  • Lots are auctioned over 3 "streets" — each street is a 30-second sealed bid round
  • Each player receives a private "hole card" (one position only they can see)
  • Additional positions are revealed to all players on Streets 2 and 3
  • All bids are revealed between streets, creating an information spiral
  • At least one "burn card" position stays hidden until the final reveal
  • The highest final-street bidder wins; profit = true value minus winning bid
  • Rooms persist between lots — players can join/leave freely

1.4 Game Currency

2. Game Rules

2.1 Auction Structure — Three Streets

  • Street 1 — "The Peek": Players see category, payout range, and their private hole card. 30-second sealed bid. All bids revealed after.
  • Street 2 — "The Flip": One new position revealed to all. Players know their hole card + 1 shared + all Street 1 bids. 30-second sealed bid. All bids revealed.
  • Street 3 — "The River": One more position revealed. Final 30-second bid. Highest bid wins. No early win.
  • The Reveal: All 7 positions exposed (including burn cards). True value calculated. Winner's profit determined.

2.2 First-Place Win Mechanic

  • Street 1: If highest bid exceeds second-highest by ≥50%, first place wins immediately. Skip to reveal.
  • Street 2: If highest bid exceeds second-highest by ≥20%, first place wins immediately.
  • Street 3: No first-place win check. Normal highest-bid auction.

2.3 Bidding Rules

  • Independent Bids: Each street is a fresh bid. Can go up or down freely.
  • Bids Revealed: All bids are shown to all players between streets.
  • $0 = Pass: Not a permanent fold. Can bid on later streets.
  • Timeout Default: No bid within 30 seconds defaults to $0.
  • Tie Breaking: Ties broken by earliest submission timestamp.
  • Overbidding Allowed: Bidding above true value results in negative profit (winner's curse).

2.4 Scoring

  • Profit Formula: Profit = True Value - Winning Bid
  • Winner Only: Only the winning bidder scores; non-winners score 0
  • Cumulative Scoring: Session leaderboard tracks cumulative profit across lots
  • Negative Profit: Overbidding results in negative profit
  • First-Place Win: Displayed with a "FIRST PLACE WIN" badge at reveal

2.5 Lot Composition

Each lot always contains exactly 7 prediction market positions:

  • Private positions (hole cards): 1 per player, dealt at start — only that player sees it
  • Shared positions: Revealed to all on Streets 2 and 3
  • Burn cards: At least 1 position stays hidden until reveal — makes lots unsolvable
  • Category: Politics, macro, culture, sports, geopolitics, emerging markets, entertainment
  • Payout Range: A deliberately wide bracket (2-3x wider than true value position)

2.6 Lot Terminology

Glossary: entry and currentPrice are prediction market probabilities (0.00-1.00). size is dollar notional (the amount staked on that position). True value = sum of (currentPrice - entry) × size across all 7 positions. Static snapshot at curation time.
